Чтобы проверить как работает сайт загрузите своё фото или воспользуйтесь фотографиями из папки "фото для теста"

Простой инструмент: закидывается фото, модель сама решает, нужно ли
осветлить, добавить контраста или насыщенности, и применяет правку.
Всё в браузере, ничего никуда не загружается.

Поддерживает JPG, PNG, BMP, HEIC. Фото до 15 Мпк, крупнее сжимаются
перед обработкой. Обработка идёт в фоновом потоке.

https://ignkos.github.io/Photo/
